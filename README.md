# import.js

It is JavaScript to import JavaScript files.
When a pass of import.js is right, that's imported according to the hierarchy.

## Usage

### HTML

```html
<script type="text/javascript" src="js/import.js"></script>
```

### JavaScript

```javascript
require('external.js');
```
